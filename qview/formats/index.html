Supported image formats

Updated for qView 4.0

The items marked "✔️ Yes" work out of the box and are included.

The items marked "❌ No" do not work out of the box and are not included. Anything listed here should have a plugin for it that you should be able to install if necessary (although it will probably be pretty involved). If you are on Arch Linux check the AUR for these plugins.

The items marked "📦 Optional" for Linux should work with packages available through your package manager (and listed as an optional dependency in the package).
